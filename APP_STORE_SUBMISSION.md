### Understanding Service Rotation
- **0-0-2 Start:** Matches always start with the "second server"
- **Server 1:** First server for the serving team
- **Server 2:** Second server (doubles only)
- **Side-Out:** When receiving team wins, serve switches to them
- **Even/Odd:** Server stands on right when team score is even, left when odd

### Using Undo
If you make a mistake:
1. Tap the "Undo" button

### Ending a Match

- Points awarded on every rally

- **Freeze-at-20 Variant:** Alternative rec mode that freezes scoring at 20+ points
